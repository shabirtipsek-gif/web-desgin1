Hover Animations

Hover Animation CSS Example

.btn-hover:hover {
  background-color: var(--palette-4);
  transform: translateY(-5px);
  box-shadow: 0 10px 20px rgba(0,0,0,0.2);
}

Click Animation CSS Example

.btn-click:active {

  transform: scale(0.95);
}
